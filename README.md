## Build Setup:
npm install or yarn add

## Download repository:
git clone https://github.com/rakhimovkamran/vuejs-budgety.git

## Go to the app:
cd vuejs-arithmetic-trainer

## Install dependencies:
npm install or yarn add

## Server with hot reload at 
http://localhost:8081/
npm run dev

## Output will be at dist/ folder
npm run build
